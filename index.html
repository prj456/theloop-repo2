The Loop Kodi Repository

Welcome! Use this repository to install The Loop addon on Kodi.

Installation Instructions

  1. Open Kodi and go to Settings → File Manager → Add Source
  2. Click "<None>" and enter this URL exactly:
    https://prj456.github.io/theloop-repo2/
  3. Name the source anything you like (e.g., The Loop Repo) and click OK
  4. Go to Add-ons → Install from Zip File
  5. Select the source you just added, then choose repository.theloop.zip to install the repository
  6. After installation, go to Install from Repository → The Loop Repository → Video Add-ons → The Loop to install the addon
